  - q: H13 트루 헤파와 일반 헤파 필터의 차이는 무엇인가요?
    a: >-
      H13은 EN 1822 유럽 표준 기준 0.3μm 입자를 99.95% 이상 차단하는 트루 헤파 등급입니다. 일반 "헤파"
      또는 "헤파 타입" 표시 제품은 H10～H11 수준(85～95% 차단)에 머무는 경우가 많아 미세먼지 PM2.5와
      0.3μm 이하 초미세 입자 차단 성능이 떨어집니다. 환경부 실내 미세먼지 가이드는 H13 이상 등급 트루 헤파를
      권장하며, 6～12개월 단위 교체가 표준입니다. 필터 등급 표시 없는 제품은 등급 외 일반 필터로 가정하고
      구매 결정을 내리는 것이 합리적입니다.
  - q: 공기청정기 평수는 어떻게 맞춰야 하나요?
    a: >-
      한국소비자원 비교 조사 기준 공기청정기는 표시 사용 면적의 1.5배 등급을 선택하는 것이 합리적입니다. 예를
      들어 거실이 10평(약 33㎡)이라면 15평(50㎡) 이상 커버 모델을 선택해야 1시간 내 3～5회 공기 순환이
      가능합니다. 표시 면적과 동일한 등급을 구매하면 강풍 모드를 상시 가동해야 해서 소음 부담과 전기료가 증가하고
      필터 교체 주기도 단축됩니다. 원룸 5～7평은 20～30㎡ 커버 소형, 거실 15～25평은 50～80㎡ 커버 중대형이
      평수별 합리적 선택의 출발점입니다.

  - q: 공기청정기 필터 교체 주기와 비용은?
    a: >-

      필터는 1만원대(샤오미 미에어 등 호환 가능 모델 기준)로 본체 가격의 5～50% 범위에서 형성됩니다. 1～2개의

## 공기청정기 추천, 한 줄로 정리하면?

공기청정기 추천 디시 인갤·클리앙·뽐뿌·더쿠 후기를 종합하면 핵심 기준은 **평수 매칭(사용 면적의 1.5배 권장) + H13 트루 헤파 필터 + CADR 수치 + 소음 dB(취침 35dB 이하)** 네 가지로 좁혀진다. "초미세먼지 99.9% 제거"라는 광고 표현은 CADR 수치와 필터 등급을 확인하지 않으면 실제 사용 환경에서 체감이 크게 떨어질 가능성이 높다. 원룸·자취 5～10평 소형이 필요하다면 슈어홈 4세대 가정용 공기청정기(179,800원), 거실 15평급 중대형이 필요하다면 ROUNY H13 트루헤파 50㎡(129,800원), 기존 샤오미 미에어 사용자 호환 필터로 유지 비용을 줄이려면 상상그램 호환 필터(12,020원)가 가성비 기준 합리적이다.

이 글은 평가 기준 → 등급표 → 추천 3선 비교 → 디시·클리앙 후기 패턴 → 환경부·한국소비자원 가이드 순으로 정리한다.

## 평가 기준 — 평수·H13 헤파·CADR·소음

공기청정기 추천 글 대부분이 "초미세먼지 잘 잡힘·조용함" 정도의 정성적 평가에 그친다. 그러나 객관적 기준은 **표시 사용 면적 대비 1.5배 매칭 + H13 트루 헤파 등급 + CADR 수치(㎥/h) + 취침 모드 소음 dB + KC 안전 인증 / CA 인증 표시**다.

| 평가 항목 | A등급 기준 | B등급 기준 | C등급 기준 |
|---|---|---|---|
| 평수 매칭 | 사용 면적의 1.5배 커버 | 사용 면적과 동등 | 사용 면적보다 작음 |
| 필터 등급 | H13 트루 헤파 명시 | 헤파 표시 (등급 X) | 필터 등급 미표시 |
| CADR | 사용 면적 ㎡ × 7.6 이상 | 5～7.6 사이 | 5 미만 |
| 소음 (약풍) | 30dB 이하 (취침 가능) | 30～40dB (거실) | 40dB 초과 |
| 인증 | KC + CA(공기청정협회) 동시 | KC 단독 | 인증 표시 없음 |

[환경부 실내 공기질 관리 가이드](https://www.me.go.kr)는 H13 등급 트루 헤파를 미세먼지 PM2.5와 0.3μm 이하 초미세 입자 차단에 권장한다. EN 1822 유럽 표준 기준 H13은 0.3μm 입자 99.95% 이상 차단, H14는 99.995% 이상 차단을 의미한다. 일반 "헤파 타입" 표시는 H10～H11 수준(85～95%)에 머무는 경우가 많아 등급 표시 확인이 필요하다.

[한국소비자원 가정용 공기청정기 비교 조사](https://www.kca.go.kr)에 따르면 표시 사용 면적과 동일한 등급을 구매하면 강풍 모드를 상시 가동해야 해 소음·전기료·필터 교체 주기 모두 부담이 커진다. 사용 면적의 1.5배 등급을 선택해야 약풍～중풍 모드 위주로 1시간 내 3～5회 공기 순환이 가능하다.

마지막으로 환경부·한국소비자원 권장을 다시 강조한다. 필터 등급·CADR·면적 매칭을 확인하지 않은 가격 단독 비교는 1～2년 내 필터 교체 비용 누적과 체감 효과 부족으로 이어지기 쉽다. H13 트루 헤파 + 면적 1.5배 + CA·KC 인증 세 가지를 패키지에서 직접 확인하는 것이 가장 현실적인 가성비 공기청정기 선택법이다.
