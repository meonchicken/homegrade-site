## 무선청소기 추천, 한 줄로 정리하면?

무선청소기 추천 디시 인갤·클리앙·뽐뿌·더쿠 후기를 종합하면 핵심 기준은 **BLDC 모터(흡입력·수명) + 흡입력 Pa 수치 + 배터리 사용 시간 + 무게(1.5kg 이하)** 네 가지로 좁혀진다. "다이슨 대체 가성비"라는 광고 표현은 모터 종류와 흡입력 스펙을 확인하지 않으면 6개월 내 흡입력 저하로 실망할 가능성이 높다. 10만원대 가성비 스틱이 필요하다면 차이슨 BLDC 무선 청소기(137,800원)나 비브르 V25000 BLDC 스틱(138,000원), 책상·차량·키보드 보조 청소용 핸디라면 디에스가추몰 충전식 무선 핸디(19,140원)가 가성비 기준 합리적이다.

이 글은 평가 기준 → 등급표 → 추천 3선 비교 → 디시·클리앙 후기 패턴 → 한국소비자원·환경부 가이드 순으로 정리한다.

## 평가 기준 — BLDC 모터·흡입력·배터리·무게

무선청소기 추천 글 대부분이 "흡입력 강하다·가볍다" 정도의 정성적 평가에 그친다. 그러나 객관적 기준은 **BLDC 모터 표시 + 흡입력 Pa 수치 + 배터리 사용 시간 + 본체 무게 + 헤파 필터 등급**이다.

| 평가 항목 | A등급 기준 | B등급 기준 | C등급 기준 |
|---|---|---|---|
| 모터 | BLDC(Brushless DC) 명시 | 카본 브러시 모터 (일반) | 모터 사양 미표시 |
| 흡입력 | 18,000Pa 이상 (또는 100AW+) | 10,000～18,000Pa | 10,000Pa 미만 (핸디) |
| 배터리 | 약함 50분+·강함 15분+ | 약함 30～50분 | 약함 30분 미만 |
| 무게 | 1.3kg 이하 (스틱) / 600g 이하 (핸디) | 1.3～1.7kg / 600～900g | 1.7kg 초과 / 900g 초과 |
| 헤파 필터 | H11 이상 등급 표시 | 헤파 필터 표시 (등급 X) | 필터 등급 미표시 |

[한국소비자원 무선청소기 비교 조사](https://www.kca.go.kr)에 따르면 BLDC 모터 제품은 일반 카본 브러시 모터 대비 평균 수명이 약 3～5배 길고, 12～24개월 사용 후에도 흡입력 저하가 10～20% 이내로 유지된다. 카본 모터 제품은 6～12개월 사용 후 흡입력이 평균 30～40% 감소한다.

[환경부 실내 미세먼지 관리 가이드](https://www.me.go.kr)는 헤파 H11 이상 등급 필터를 권장하며, 미세먼지 PM2.5와 집먼지 진드기 차단에 효과적이라고 안내한다. 필터는 6～12개월 단위 교체가 권장되며, 교체 비용은 가성비 라인 기준 1～2만원대다.
